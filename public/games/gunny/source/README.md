# Gunny

Small browser space shooter. Third-person ship. Earth, moon, stars, enemy raiders, crashy satellites.

## Run

```bash
npm install
npm run dev
```

Open the local Vite URL.

## Controls

- `WASD` or arrow keys: steer
- `Space`: fire
- Mouse hold: fire
- Touch buttons: mobile steer + fire

## Goal

Drop 12 raiders before your hull hits zero.
